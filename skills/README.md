# 🧠 刘德超财经分析 Skill

基于刘德超（观棋有语）748个抖音视频提取的认知框架和思维模型。

## 📊 数据概览

- **视频数量**: 748个
- **数据时间范围**: 2023-08-06 至 2026-04-15
- **心智模型**: 123个
- **决策启发式**: 70条
- **提取方法**: Claude Opus 4.7 + 三重验证（跨域复现、生成力、排他性）

### 方法3: API集成

```python
import json

# 加载认知框架数据
with open('cognitive_synthesis.json', 'r', encoding='utf-8') as f:
    framework = json.load(f)

# 查询特定主题的心智模型
def get_models_by_domain(domain):
    return [
        m for m in framework['mental_models']
        if domain in m.get('domains', [])
    ]

# 示例：获取地缘政治相关的模型
geopolitics_models = get_models_by_domain('地缘政治')
```

## 📊 质量指标

### 提取质量
- **成功率**: 95.1% (711/748个视频成功提取)
- **平均每视频**: 3-4个心智模型，3条决策启发式
- **跨域验证**: 所有心智模型至少在2个不同领域复现

### 验证标准
所有心智模型通过三重验证：
1. **跨域复现**: 在至少2个不同话题中出现
2. **生成力**: 可用于推断新情况的立场
3. **排他性**: 体现个人独特视角，非通用常识

### 置信度分布
- 高置信度 (>0.8): 45个模型
- 中置信度 (0.6-0.8): 58个模型
- 低置信度 (0.4-0.6): 20个模型

## ⚠️ 使用限制

**这个Skill能做到的**：
- ✅ 用刘德超的分析框架解读经济形势
- ✅ 识别他关注的关键变量和分析路径
- ✅ 模拟他的表达风格和语气

**这个Skill做不到的**：
- ❌ 无法预测刘德超对全新话题的立场（超出748个视频范围）
- ❌ 无法替代他的创造力和即时判断
- ❌ 公开表达 vs 真实想法可能有差距
- ❌ 信息截止到 2026-04-15
