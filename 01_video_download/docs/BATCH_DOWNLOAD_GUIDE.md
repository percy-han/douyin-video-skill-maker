## 🔧 常见问题

### Q1: Cookie过期了怎么办？

**现象**：下载到一半突然失败，提示Cookie无效

**解决**：
```bash
# 重新配置Cookie
python3 setup_cookie.py

# 继续下载（F2会自动跳过已下载的视频）
python3 batch_download.py
```

### Q2: 下载中断了怎么办？

**解决**：直接重新运行`python3 batch_download.py`

F2会自动检查已下载的文件，跳过它们，只下载缺失的视频。

### Q3: 部分视频下载失败？

**原因**：
- 视频被删除
- 视频设置为私密
- 网络波动

**解决**：查看日志，确认失败原因。大部分情况下重试即可。

### Q4: 内存不足？

**现象**：Whisper转录时内存占用过高

**解决**：使用更小的模型
```bash
python3 whisper_transcribe.py batch_output/ --batch --model small
```

### Q5: 下载速度慢？

**原因**：
- 网络限速
- F2默认并发数较低

**优化**：修改`batch_download.py`中的参数
```python
"-t", "10",  # 将并发任务数从5改为10（谨慎使用，可能被限流）
```

## 💡 优化建议

### 1. 分批下载（推荐）

如果视频太多（如500+），建议分批下载：

```bash
# 限制下载数量为100个
python3 -m f2 dy -M post \
  -u "博主主页链接" \
  -k "$(cat douyin_cookie.txt)" \
  -o 100 \
  -p batch_output_1/

# 然后修改输出目录继续下载下一批
```

### 2. 使用代理（可选）

如果下载速度慢或被限流：

```bash
# 修改batch_download.py，添加代理参数
"-P", "http://127.0.0.1:7890", "http://127.0.0.1:7890"
```

### 3. 断点续传

F2和Whisper都支持断点续传，中断后重新运行即可。
