# 🎤 视频转文字方案对比

## 两个方案

### 方案A：Whisper（开源，OpenAI）
- 本地或EC2上运行
- 需要GPU加速（可选）
- 一次性成本

### 方案B：Amazon Transcribe（AWS托管）
- API调用，按使用量付费
- 无需GPU
- 零维护

### 2. 成本对比

#### Whisper成本（EC2运行）

**EC2实例成本**（按需定价）：

| 实例类型 | GPU | 价格/小时 | 100小时视频转录成本 | 推荐度 |
|---------|-----|-----------|-------------------|--------|
| g4dn.xlarge | T4 (16GB) | $0.526 | $52.60 | ⭐⭐⭐ 推荐 |
| g5.xlarge | A10G (24GB) | $1.006 | $100.60 | ⭐⭐ 速度快 |
| t3.xlarge | 无GPU | $0.1664 | $16.64 | ⭐ 极慢 |

**转录速度**：
- GPU (g4dn): 1小时视频 → 3-5分钟转录
- CPU: 1小时视频 → 2-3小时转录

**100小时视频预估**：
- g4dn.xlarge: 5-8小时运行 → **$4-5**
- t3.xlarge (CPU): 200-300小时 → **$33-50**（不推荐）

#### Amazon Transcribe成本

**定价**：
- 标准：$0.024/分钟（$1.44/小时）
- 批量：$0.0144/分钟（$0.864/小时）前12个月

**100小时视频预估**：
- 标准定价：100小时 × $1.44 = **$144**
- 批量定价：100小时 × $0.864 = **$86.4**（前12个月）

### 5. 易用性对比

#### Whisper

```bash
# 优点：一行命令搞定
whisper video.mp4 --language Chinese --model large

# 缺点：需要管理EC2、安装依赖
pip install openai-whisper
```

#### Amazon Transcribe

```python
# 优点：AWS原生集成，无需管理基础设施
import boto3
transcribe = boto3.client('transcribe')

# 缺点：需要上传到S3，多一步操作
s3.upload_file('video.mp4', 'bucket', 'video.mp4')
transcribe.start_transcription_job(...)
```
